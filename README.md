## 主な機能

- ポケモン一覧取得
- ポケモン検索機能（リアルタイム）
    - 名前（テキスト）
    - タイプ（混合タイプ含み）
- レスポンシブ対応

```bash
# リポジトリをクローン
git clone https://github.com/Keita-0025/Pokemon-pictorial-book.git

cd リポジトリ名

# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev
